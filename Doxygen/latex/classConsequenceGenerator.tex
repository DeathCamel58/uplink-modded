\doxysubsection{Detailed Description}
Consequence Generator. 

This is responsible for analysing an action that has happened, and generating subsequence \char`\"{}consequence\char`\"{} actions.

Example action\+: A user is caught performing a hack Example consequence\+: User\textquotesingle{}s Uplink rating is decreased, and they receive a fine

\begin{DoxyNote}{Note}
This class should operate in D\+A\+TA only. It shouldn\textquotesingle{}t be handling screen flashes, interface changes, etc. Ideally, it should stick to the world module. 
\end{DoxyNote}
