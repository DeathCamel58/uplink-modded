Detailed Description

Consqeuence Generator.

This is responsible for analysing an action that has happened, and generating subsequence "consequence" actions.

Example action: A user is caught performing a hack Example consequence: User's Uplink rating is decreased, and they receive a fine

Note
This class should operate in DATA only. It shouldn't be handling screen flashes, interface changes, etc. Ideally, it should stick to the world module.
